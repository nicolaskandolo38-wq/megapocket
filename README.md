# MegaPocket — Business Suite

## 🚀 Démo immédiate
Ouvrir `index.html` dans n'importe quel navigateur.

## 📱 Installer comme app (PWA)
1. Ouvrir dans Chrome/Edge sur Android
2. Menu → "Installer l'application" ou "Ajouter à l'écran d'accueil"
3. Fonctionne **offline** — toutes les données sont stockées localement

## ⚡ Fonctionnalités
- 🧾 **Factures** — Créer, consulter, imprimer (PDF)
- 👥 **Clients** — Gestion complète (CRUD)
- 📦 **Stock** — Inventaire avec alertes quantité
- 📱 **QR Code** — Carte de visite digitale, téléchargeable
- 🌙 **Dark/Light** — Thème automatique
- 💾 **Offline-first** — IndexedDB, tout fonctionne sans internet
- 📤 **Export/Import** — Sauvegarde JSON des données
- 📲 **Installable** — PWA, s'installe comme une vraie app

## 🛠️ Stack
- HTML5 + CSS3 + JavaScript vanilla
- IndexedDB (stockage local)
- Service Worker (offline)
- PWA Manifest (installable)
- Zéro dépendance externe (sauf Font Awesome CDN)

## 📊 Données de démo
3 clients, 6 produits, 4 factures préchargés à la première ouverture.
